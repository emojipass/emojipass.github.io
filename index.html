Welcome to emojipass! This is a website created by HCS Group AD to demonstrate a comparison of login functionality with pins using either digits or emojis.

The system has been implemented to mimic a real website, to evaluate both the usability and memorability of both emoji and digit pins. However, this is just a mockup - account access and registration is simulated using localstorage, and an external Firebase database is used with a one-way connection so that experiment facilitators can add participant details so they can attempt login at home.

The Experiment

An experiment facilitator is able to toggle the site between either using the emoji pin condition or the digit pin condition using an admin interface which is hid during the experiment. This allows us to easily compare the two conditions in the same setting without relying on seperate deployments. As the experiment will likely be conducted on participants' own devices, and these configurations are stored in localstorage, a participant should be able to access the same configuration from home without further input from the experiment facilitators.

During the experiment, data will be collected on success or failure during logins with the associated condition, and following this, a survey will be used to guage participants' opinions and feelings towards the systems respectively.
